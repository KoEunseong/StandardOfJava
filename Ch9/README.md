Chapter9 java.lang 패키지와 유용한 클래스
======================
## 1. java.lang 패키지
**java.lang** 패키지는 자바에서 가장 기본이 되는 클래스들을 포함하고 있고 그렇기 때문에 import 없이도 사용이 가능하다.(ex String, System 클래스)


### 1.1 Object클래스
Object클래슨는 모든 클래스의 최고 조상이기 때문에 Object클래스의 멤버들은 모든 클래스에서 바로 사용 가능하다.
